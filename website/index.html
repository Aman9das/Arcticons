Arcticons

A monotone line-based icon pack for Android.

  • Download
  • Read more

A line-based icon pack forked from Frost but rebuild with consistent sizing and adjustments specifically for lines in mind. Arcticons is FOSS and provides mostly icons for open-source apps on F-Droid (but there are many other apps in this pack too). The design is simple and clean with minimal lines that give you a nice see-trough effect.

Two versions for the dark- and lightmode user

Contribute

If an icon for your app is missing, the easier way for request it is using Turtl app in order to find the package name of the app. After, you can open an issue with these data. Check details on GitLab.

If you want to, you can draw by yourself the icon, and open a merge request. Again, detailed instructions are on GitLab.
